# AI Filter 设计资源

## 1. 设计语言

本项目视觉风格对标**飞书多维表格（Lark Base）**，采用清晰、高效、信息密度高的企业级 UI 设计。

### 使用规范
1. **新增或修改 UI 组件时**，必须先从 Figma 组件库获取对应组件的最新设计规范（尺寸、颜色、间距、状态）
2. **新增或修改图标时**，必须先从 Figma 图标库获取对应图标的 SVG 路径和尺寸
3. **设计与代码不一致时**，以 Figma 设计稿为准
4. 通过 Figma MCP 工具（`get_design_context`、`get_screenshot`）获取设计资源

## 9. 交互规范

### 鼠标交互
| 交互 | 行为 |
|------|------|
| 单击单元格 | 选中（蓝色边框） |
| 双击单元格 | 进入编辑态 |
| 单击已选中单元格 | 进入编辑态 |
| 拖拽单元格 | 矩形选区（4px 阈值触发） |
| 拖拽列头 | 列排序（4px 阈值触发） |
| 拖拽列边界 | 调整列宽 |
| 右键行 | 上下文菜单 |
| 右键列头 | 列上下文菜单 |

### 键盘交互
| 快捷键 | 行为 |
|--------|------|
| `Delete` / `Backspace` | 清空选中单元格 / 清空选中行单元格 |
| `Escape` | 取消选区 / 关闭面板 |
| `Enter` | 提交编辑 / 提交 AI 查询 |
| `Ctrl+Z` / `⌘+Z` | 撤销 |
| 长按 `Space` (500ms) | 语音输入 |
| `Shift+Click` | 行范围选择 |

### 状态反馈
| 状态 | 表现 |
|------|------|
| 行悬浮 | 背景变灰，显示复选框 |
| 行选中 | 蓝色背景，复选框勾选 |
| 单元格选中 | 蓝色边框 + 淡蓝背景 |
| 编辑态 | 白色背景 + 蓝色边框 + 内容可编辑 |
| Loading | 动态省略号 + SparkleIcon 动画 |
| 录音中 | 麦克风图标变色 + 脉冲动画 |
